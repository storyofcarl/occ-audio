# Prompt Guide

## Text Prompt to Audio \(T2A\)

Good text prompt should include:

1. Detailed environment description \(weather/location/context/\.\.\.\)

2. Detailed background music/sound effects description \(classic orchestral/hiphop/high heels/chuckles/\.\.\.\)

3. Character action/appearance description \(waving hands/playing soccer/\.\.\.\)

4. Character voice description \(gender/race/age/emotion/tone/speed/\.\.\.\)

5. What characters are going to say\.
